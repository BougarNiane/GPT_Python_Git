📋 네이버 자동 로그인 프로그램 사용 가이드
=============================================

🚀 빠른 시작
-----------
1. install.bat 실행 → 라이브러리 설치
2. run_naver_login.bat 실행 → 프로그램 실행
3. GUI에서 아이디/비밀번호 입력 후 로그인

📁 파일 설명
-----------
• naver_auto_login.py          - 기본 버전
• naver_auto_login_advanced.py - 고급 버전 (권장)
• install.bat                  - 라이브러리 자동 설치
• run_naver_login.bat          - 프로그램 실행
• requirements.txt             - 필요한 라이브러리 목록
• README.md                    - 상세 사용법

🔧 주요 기능
-----------
✅ GUI를 통한 간편한 로그인
✅ Chrome/Edge 브라우저 지원  
✅ 아이디 저장 기능
✅ 백그라운드 실행 (헤드리스 모드)
✅ 자동 브라우저 닫기
✅ 설정 자동 저장/불러오기
✅ 자동 드라이버 관리

⚙️ 고급 옵션
-----------
• 헤드리스 모드: 브라우저 창 없이 백그라운드 실행
• 자동 닫기: 로그인 성공 후 브라우저 자동 종료
• 아이디 저장: 다음 실행 시 아이디 자동 입력

🛡️ 주의사항
-----------
• 개인 사용 목적으로만 사용
• 네이버 이용약관 준수
• 과도한 사용 금지
• 2단계 인증 해제 필요

❓ 문제 해결
-----------
• 로그인 실패 → 아이디/비밀번호 확인
• 드라이버 오류 → 고급 버전 사용 권장
• 실행 오류 → 관리자 권한으로 실행

📞 지원
-------
문제가 발생하면 README.md 파일의 문제 해결 섹션을 참고하세요.
